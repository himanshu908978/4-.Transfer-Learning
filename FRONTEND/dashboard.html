About this project

This is my forth project in the domain of AI/ML/DL whose AI model is made by using the Pytorch and concepts of CNN(Convulational Neural Network) and trained on Dog vs Cat classification dataset on Kaggle.

This Image Classification project detects dog and cat in your given photo but you will get choice to select one of the pretrained model in the three models that we fine-tuned in this project.

This trained AI model detect the labels(Dog or Cat) in image and gives you a Number(label) that it could find in your given image and also give a Confidence score that represents that how much confident model really is to give you that output.

Labels :-

  • cat
  • dog

So if you upload a image that belongs to any of these label's category then you will get a predicted class from model in this scenario dog and cat label and confidence score and confidence score can vary on your choice of model.

Conclusion

I trained this model on google colab notebook's free GPU T4(it comes with a specfic duration of usage) for 5 epochs/model so total 15 epochs and then

VGG :-

  • Loss :- 0.37
  • Accuracy on testing dataset :- 0.97

Resnet :-

  • Loss :- 0.45
  • Accuracy on testing dataset :- 0.985

EfficientNet :-

  • Loss :- 0.7
  • Accuracy on testing dataset :- 0.96

This accuracy is preety good and also the cause of this accuracy is obviously our pretrained model models like VGG, Resnet and EfficientNet are already trained on a very large amount of parameters and data, so model have already see a variety of data.
